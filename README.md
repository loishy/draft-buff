## Draft Buff

It be created to help vim user easily create new buffer with chosen language for testing or use it as presentation

## Supported Languages

- Ruby
- Typescript
- Javascript
- Golang
- Java
- Python
- Rust
- C
- C++
- Swift
- Haskell
- Scala

## Installation

```lua
use {
  'loishy/draft-buff',
  requires = {
    'MunifTanjim/nui.nvim'
  },
}
```

## Usage

```lua
:lua require"draft-buff'.select_lang()
```
