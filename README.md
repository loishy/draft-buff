## Draft Buff
It be created to help vim user easiliy create new buffer with choosen language for testing or use it as presentation

## Support Languages
- Ruby
- Typescript
- Javascript
- Golang
- Java
- Python
- Rust
- C
- C++
- Haskell
- Scala

## Installation

```lua
use {
  'loishy/draft-buff',
  requires = {
    'MunifTanjim/nui.nvim'
  },
}
```

## Usage

```lua
:lua require"draft-buff'.select_lang()
```
